== Description ==
<a href="http://www.surfpeople.net/">Surfpeople</a> - Live Community is a social networking service that allows members to create unique personal profiles online in order to find and communicate with old and new friends.

This plugin adds a small widget within each  blog  . Displays Activity live feed about your Surfpeople Profile and increment your visibility in the world .

<h4>Try the Demo</h4>

If you're interested in seeing what a default installation of Social Widget  has to offer, try out the Social Widget  Test Blog! 

<a href="http://www.surfpeople.net/blog/">Social Widget  Test Blog </a>

== Installation ==

Installation is fairly simple

1. Upload `widget_surfpeople.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Active Surfpeople in the Apparence - Widget and insert your Id Surfpeople.
4. Surfpeople is free Social Network <a href="http://www.surfpeople.net/join.php">Join Now </a>
